Для того, чтобы отредактировать пару нужно на странице Расписание на соответствующей паре нажать на кнопку в левом верхнем углу ячейки таблицы. Кнопка появляется в ячейках только при наведении на них. Рисунок 1 - Кнопка для редактирования пары
После нажатия на кнопку появится модальное окно, сорежащее три поля для заполнения:
  • Дисциплина
  • Группа
  • Аудитория
Все три поля для ввода обязательны для заполнения, а также имеют ограничения на количество допустимых символов:
  • Дисциплина - 50 символов
  • Группа - 20 символов
  • Аудитория - 14 символов
Кроме этого внизу модального окна есть специальный "флажок" Оставить ячейку пустой. Если в нём поставить галочку, то все поля для ввода будут заблокированы, а сама ячейка таблицы при сохранении данных останется пустой.
Если поставить флажок Оставить ячейку пустой после того, как данные были сохранены, то ранее введённые в поля данные останутся, хоть и не будут отображаться с таблице. Рисунок 2 - Модальное окно с введёнными данными Рисунок 3 - Модальное окно с некоррестными данными (превышение допустимого числа символов, пустое поле) Рисунок 4 - Модальное окно с поставленным флажком Оставить ячейку пустой Для сохранения данных можно воспользоваться кнопкой Сохранить , находящейся в нижнем правом углу модального окна или же клавишей "Enter" на клавиатуре. Кнопка Сохранить внизу модального окна блокируется в моменты, когда какие-то данные введены неверно. Клавиша "Enter" в таких случаях также не сработает. В любой момент времени работы с модальным окном его можно закрыть. Это можно сделать, нажав кнопку в правом верхнем углу модального окна или же по клавише "Escape" на клавиатуре.
