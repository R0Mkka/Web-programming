Для удаления дисциплины нужно для начала на странице Дисциплины выбрать и открыть её. Для этого будет достаточно один раз кликнуть по соответвующей дисциплины в списке. После этого откроется страничка с её содержимым. Рисунок 1 - Пример первоначального экрана для дисциплины Следующим шагом необходимо нажать на кнопку в правом верхнем углу страницы. Рисунок 2 - Кнопка для удаления дисциплины После этого появится модальное окно для подтверждения действия. Рисунок 3 - Модальное окно для подтверждения удаления дисциплины Последним этапом, после нажатия на кнопку Да приложение автоматически переместит Вас на страничку Дисциплины. Если операция удаления прошла успешно, то соответсвующей дисциплины в списке быть не должно.
