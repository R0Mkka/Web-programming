Для добавления дисциплины нужно на странице Дисциплины нажать на кнопку add menu_book в правом верхнем углу страницы. Рисунок 1 - Кнопка для добавления дисциплин
После нажатия на кнопку add menu_book появится модальное окно, содержащее три поля для ввода:
  • Название дисциплины
  • Описание дисциплины
  • Дисциплина для курса
Все три поля обязательны для заполнения и имеют некоторые ограничения на ввод:
  • Название дисциплины - максимум 50 символов
  • Описание дисциплины - максимум 255 символов
  • Дисциплина для курса - допустим только числовой ввод, диапазон от 1 до 6
 Рисунок 2 - Модальное окно добавления дисциплин
В любой момент работы с модальным окном его можно закрыть с помощью кнопки в правом верхнем углу модального окна.
После заполнения всех полей корректными данными для создания дисциплины нужно нажать на кнопку Добавить в нижнем правом углу модального окна. Сразу после этого приложение автоматически перенаправит Вас на страничку только что созданной дисциплины. По умолчанию это страничка Программа курса.
В случае ввода некорректных данных, кнопка Добавить будет заблокирована. Рисунок 3 - Страничка новой дисциплины
