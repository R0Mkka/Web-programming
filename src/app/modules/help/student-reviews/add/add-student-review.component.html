Для добавление отзыва необходимо на странице Отзывы студентов нажать на кнопку в верхнем правом углу рабочей области (рисунок 1). Рисунок 1 - Кнопка Добавить
После нажатия на кнопку появится модальное окно с тремя полями:
  • Полное имя студента
  • Группа студента
  • Текст отзыва
Все три поля обязательны для заполнения и имеют ограничения на количество символов:
  • Полное имя студента - 50 символов
  • Группа студента - 18 символов
  • Текст отзыва - 256 символов
В любой момент процесса заполнения полей модального окна оно может быть закрыто по нажатию кнопки в правом верхнем углу или клавиши "Escape" на клавиатуре.
Для сохранения отзыва необходимо нажать на кнопку Добавить или на клавишу "Enter" на клавиатуре. На рисунке 2 представлен пример формы с корректно введёнными данными.
В случае ввода некорректных данных кнопка Добавить будет заблокирована.
