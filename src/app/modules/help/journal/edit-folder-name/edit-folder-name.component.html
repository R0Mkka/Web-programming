Для создания папки необходимо на странице Журнал нажать на кнопку add folder в верхнем правом углу рабочей области (рисунок 1). Рисунок 1 - Кнопка Создать папку
После этого на экране появится модальное окно с двумя полями: название папки и доступность папки. Первое поле обязательно для заполнения и имеет ограничение в размере на 40 символов. Второе поле - это выпадающий список, содержащий следующие значения:
  • Полный доступ
  • Только для чтения
  • Приватная
Значением по умолчанию является Приватная. После создания папки она автоматически появится в соответсвующей секции.
Рисунок 2 - Создание новой папки Создать папку можно по нажатию на кнопку Создать или по нажатию клавиши "Enter" на клавиатуре. Рисунок 3 - Результат создания новой папки В любой момент процесса заполнения полей модального окна оно может быть закрыто по нажатию кнопки в правом верхнем углу или клавиши "Escape" на клавиатуре.
