Для изменения названия папки нужно для начала на странице Журнал выбрать и открыть её. Для этого будет достаточно один раз кликнуть по соответвующей папке. После этого откроется страничка с её содержимым - списком ранее созданных листов или же пустой папкой без листов. Рисунок 1 - Пример первоначального экрана для пустой папки Следующим шагом необходимо дважды кликнуть по имени текущей папки и тогда вместо обычного текста на месте названия появится поле для ввода текста. Здесь можно ввести новое имя для папки. Новое имя папки должно быть непустым и не более 30 символов в длину. Рисунок 2 - Название папки в режиме для редактирования Для сохранения нового имени папки необходимо или сбить с поля ввода фокус (т.е. кликнуть мышкой где-то на экране за пределами поля ввода), или нажать клавишу "Enter" на клавиатуре. После этого имя будет сохранено автоматически.
