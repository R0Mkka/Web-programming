Режим доступа папки можно изменить только на странице Журнал. Для этого необходимо "взять" папку, которую нужно изменить, и "перетащить" её в соответсвующий блок. Рисунок 1 - Демонстрация режима перетаскивания папки, шаг 1 Как можно увидеть на рисунке 1, когда папка будет "взята" возле курсора будет почти полностью прозрачные очертания данной папки, а сама папка будет подсвечена другим цветом. Рисунок 2 - Демонстрация режима перетаскивания папки, шаг 2 После перемещения папки в новый блок, её режим доступа будет изменён автоматически на соответсвующий.
