Для удаления папки нужно для начала на странице Журнал выбрать и открыть её. Для этого будет достаточно один раз кликнуть по соответвующей папке. После этого откроется страничка с её содержимым - списком ранее созданных листов или же пустой папкой без листов. Рисунок 1 - Пример первоначального экрана для пустой папки Следующим шагом необходимо нажать на кнопку в правом верхнем углу страницы. После этого будет отображено модальное окно, запрашивающее подтверждение на производимое действие. Рисунок 2 - Модальное окно для подтверждения удаления Для подтверждения удаления нужно нажать на кнопку Да . В то время как для отмены можно воспользоваться кнопкой Отмена или же кнопкой в правом верхнем углу модального окна. Сразу после удаления папки система автоматически перенаправит Вас на главну страницу журнала. Если всё прошло успешно, то только что удалённая папка должна отсутсвовать в списке существующих папок.
