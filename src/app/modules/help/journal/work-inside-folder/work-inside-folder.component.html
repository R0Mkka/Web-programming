  • Создание папки
  • Удаление папки
  • Изменение названия папки

Для начала работы с папкой нужно на странице Журнал выбрать соотвествующую папку и открыть её, один раз кликнув по ней.
Рисунок 1 - Пример первоначального экрана для пустой папки Основная суть работы с папкой - это работа с листами внутри неё. Листы представляют из себя что-то отдалённо напоминающее листы из программы MS Excel. Для создания или добавления нового листа в папку необходимо нажать кнопку, находящуюся чуть ниже названия папки. Рисунок 2 - Кнопка для создания нового листа После добавления листа он сразу же будет отображён в горизонтальном списке на уровне с кнопкой добавления листов. По умолчанию все новые листы называются "Новый лист", однако данное название можно изменить в любой момент времени для каждого листа по отдельности. Для этого нужно дважды кликнуть по соответсвующему листу и в появившемся поле ввести новое имя. Оно должно быть непустым и иметь длину не более 30 символов. Рисунок 3 - Изменение название листа
В каждом листе есть 4 основные функциональные кнопки:
  • Добавить ученика
  • Добавить день
  • Удалить лист
  • Сохранить изменения
 Рисунок 4 - Функциональные кнопки листа
Первые две кнопки (добавить ученика и добавить день) отвечают за расширение рабочей области листа. В данном случае имеется ввиду не конкретное добавление какой-то сущности ученика или дня, а просто добавление нового ряда или столбца в сетку соответственно. Данные названия были даны ввиду направленности приложения и построением определённых ассоциаций, так как в журнале принято горизонтально размещать учеников, а вертикально - рабочие дни.
Третья кнопка в списке необходима для удаления листа. После нажатия на данную кнопку будет показано модальное окно, запрашивающее подтверждения действия.
Рисунок 5 - Модальное окно на подтверждение удаление листа Последняя кнопка в списке, как понятно из названия, отвечает за сохранение изменений в листе. Это касается как изменений в размере рабочей области (добавление новых столбцов или рядов), так и изменений в виде данных сетки. Никакие изменения в листе не будут сохранены до нажатия кнопки save (сохранить изменения).
В ячейки сетки можно вводить любые данные. Когда ячейка становится активной, в ней появляется курсор и она подсвечивается другим цветом. Помимо точечного выбора ячеек с помощью кликов мыши, по сетке можно перемещаться с помощью клавиатуры. Для этого можно использовать следующие клавиши:
  • "Стрелочка Вверх" - Перемещение к верхней ячейке относительно текущей (если выше ещё есть ячейка)
  • "Стрелочка Вниз" - Перемещение к нижней ячейке относительно текущей (если ниже ещё есть ячейка)
  • "Стрелочка Влево" - Перемещение к левой ячейке относительно текущей (если слева ещё есть ячейка)
  • "Стрелочка Вправо" - Перемещение к правой ячейке относительно текущей (если справа ещё есть ячейка)
  • "Enter" - Перемещение к нижней ячейке относительно текущей (если ниже ещё есть ячейка)
  • "Tab" - Перемещение к правой ячейке относительно текущей (если справа ещё есть ячейка)
  • "Shift" + "Tab" - Перемещение к левой ячейке относительно текущей (если левой ещё есть ячейка)
